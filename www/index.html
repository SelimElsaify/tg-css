Antworden op vragen handout CSS

  1. Je kunt op 3 verschillende manieren CSS in je HTML inladen
    • Embedded ccs via style tag in de head tag
    • inline styling: in de tag zelf via style tag
    • External style: hier wordt alle styling in een aparte bestand geschreven en wordt deze naar de html gelinkd via de link tag
  3. CSS syntax
    • CSS heeft een apart syntax met name het gebruik van selectors, id, classes en moet volgens de regels van het w3c school geschreven en gevalideerd wordt
  4. Selectors
    • universal selector: Er wordt een * er voor gebruikt om als een algemeen styling te geven
    • Id selector: wordt met een # gemaakt en dat heeft met een ID te maken zoals div main nav en het heeft hoge prioriteit. Ex. #main{hier komt de styling te staan}, of div id="geef een id naam" die wordt in de css gestyled
    • class selector: door een . gemaakt voor de naam ex: .power{}
    • Descendent Selector: elements die descendent zijn van een element ex div p. p is een descendent van de div
    • child selector: div > p deze spreekt alle direct childern van een element
    • Adjacent Sibling Selector div + p: om een element direct na een specifieke element aan te spreken. Alle elementen moeten childern of the same parent / element
    • General sibling ~ selector: selecteert alle kleine kinderen van een specifieke element
